# ESco Client Portal V2

A modern, interactive prototype of the ESco subscription analytics portal featuring a Success Scorecard for helping clients optimize their subscription metrics.

## Features

- **Dashboard** - Real-time KPI metrics (Orders, Subscribers, Copies)
- **Success Scorecard** - 12 performance indicators across 2 categories with status indicators
- **Analytics** - Charts and trend analysis (coming soon)
- **Reports** - Report library and downloads (coming soon)
- **Settings** - Account and integration management (coming soon)

## Tech Stack

- HTML5
- CSS3 (Responsive, modern design)
- JavaScript (Vanilla, no frameworks)
- Fully responsive (mobile, tablet, desktop)

## Getting Started

### Option 1: Open in Browser
Simply open `index.html` in your web browser.

### Option 2: Use Live Server
In VS Code:
1. Install "Live Server" extension
2. Right-click `index.html`
3. Select "Open with Live Server"
